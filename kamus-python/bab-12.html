BAB 12: MENGOLAH BAHAN BAKU MENTAH

Bab 12 / Kamus Python (Data Processing)

Sebelum masakan disajikan atau diolah oleh AI, bahan-bahan mentah (Data) harus dibersihkan, diurutkan, dan disiapkan terlebih dahulu. Bab ini berisi kosa kata kunci untuk mengolah data dalam jumlah besar di Python.

📖 KATA 74: NumPy (dibaca: nam-pai / singkatan dari: Numerical Python)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Sebuah Library (alat pinjaman) ajaib di Python yang diciptakan khusus untuk menghitung jutaan angka dalam sekejap mata.

💡 ANALOGI DUNIA NYATA:

Jika kodingan Python biasa adalah "Kalkulator Warung" yang menghitung angka satu per satu, maka NumPy adalah "Kalkulator Super Komputer". Ia bisa menjumlahkan, mengalikan, atau membagi jutaan angka secara bersamaan tanpa membuat laptop Anda nge-lag.

⚠️ ATURAN WAJIB:

Harus selalu "dipinjam" di baris paling atas kodingan Anda dengan julukan standar seluruh dunia: import numpy as np.

💻 CONTOH KODE:

import numpy as np

# Membuat daftar 3 angka
harga = np.array([1000, 2000, 3000])

# Mengkalikan SEMUA angka dengan 2 secara serentak (Python biasa tidak bisa begini!)
harga_diskon = harga * 2
print(harga_diskon)

📖 KATA 75: Array (dibaca: a-rei / artinya: Susunan / Laci Angka)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Tempat penyimpanan khusus buatan NumPy. Bentuknya seperti rak bersusun yang sangat sangat rapi, khusus untuk menyimpan angka agar gampang dihitung oleh AI.

💡 ANALOGI DUNIA NYATA:

Bayangkan Rak Telur. Setiap lekukan di rak telur punya posisi yang presisi. Anda bisa punya rak telur 1 dimensi (sebaris), 2 dimensi (berbaris dan berkolom seperti papan catur), hingga 3 dimensi (seperti Rubik). Di dalam kotak-kotak inilah AI menyimpan piksel gambar atau koordinat teks.

⚠️ ATURAN WAJIB:

Berbeda dengan wadah List (Kata 10) yang bisa dicampur isi teks dan angka, sebuah Array wajib berisi tipe data yang sama persis (misal: angka semua). Kalau dicampur, NumPy akan kebingungan menghitungnya.

💻 CONTOH KODE:

import numpy as np

# Membuat rak telur (array) berisi 5 angka
rak_angka = np.array([10, 20, 30, 40, 50])

# Kita bisa memanggil angka pertama (ingat, komputer menghitung dari 0)
print(rak_angka[0]) # Akan mencetak: 10

📖 KATA 76: Pandas (dibaca: pan-das / singkatan dari: Python Data Analysis)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Library (alat pinjaman) paling terkenal di dunia Python untuk membuka, membaca, dan mengacak-acak data berbentuk tabel. Catatan: Ini sama sekali tidak ada hubungannya dengan hewan Panda 🐼.

💡 ANALOGI DUNIA NYATA:

Pandas adalah Microsoft Excel versi Koding. Apapun yang biasa Anda lakukan di Excel (buka file CSV, filter kolom, hapus baris, menggabungkan tabel), Pandas bisa melakukannya ribuan kali lebih cepat hanya dengan mengetik perintah teks, tanpa perlu nge-klik mouse sama sekali.

⚠️ ATURAN WAJIB:

Sama seperti NumPy, alat ini punya julukan standar internasional yang pantang dilanggar. Anda wajib mengetik: import pandas as pd.

💻 CONTOH KODE:

import pandas as pd

# Menyuruh Pandas membuka file Excel/CSV
tabel_karyawan = pd.read_csv("data_gaji.csv")

📖 KATA 77: DataFrame (dibaca: de-ta-freim / artinya: Bingkai Data / Tabel)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Sebutan resmi di dunia Python untuk sebuah "Tabel" yang dihasilkan oleh Pandas. Ia memiliki baris ke bawah dan kolom ke samping.

💡 ANALOGI DUNIA NYATA:

Sama persis seperti Selembar Kertas Kerja (Worksheet) di Microsoft Excel. Jika Pandas adalah aplikasi Excel-nya, maka DataFrame adalah tabel tempat Anda melihat nama-nama kolom seperti "Nama", "Umur", dan "Gaji".

⚠️ ATURAN WAJIB:

Setiap DataFrame pasti punya Index (nomor urut baris di sisi paling kiri) dan Columns (nama judul di bagian paling atas).

💻 CONTOH KODE:

import pandas as pd

# Membuat DataFrame (Tabel) sederhana
data = {"Nama": ["Andi", "Budi"], "Umur": [25, 30]}
tabel_kita = pd.DataFrame(data)

print(tabel_kita) # Akan tercetak rapi berbentuk baris dan kolom di layar

📖 KATA 78: Data Cleaning (dibaca: de-ta kli-ning / artinya: Cuci Data)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Pekerjaan kuli (namun sangat penting) untuk membuang, menambal, atau merapikan data-data yang rusak, kosong, atau salah ketik sebelum data tersebut diberikan ke AI.

💡 ANALOGI DUNIA NYATA:

Seperti Menyortir Beras. Sebelum beras dimasak di rice cooker, Anda harus membuang gabah hitam, kerikil kecil, atau kutu beras. Kalau tidak dicuci (cleaning), nasinya akan tidak enak dan bisa membuat gigi patah. Di AI, data yang tidak dicuci akan membuat model AI bertingkah bodoh.

⚠️ ATURAN WAJIB (Hukum Besi AI):

Garbage In, Garbage Out (GIGO). Sampah yang masuk, Sampah pula yang keluar. Sebagus apapun otak AI yang Anda pakai, kalau datanya kotor, jawabannya pasti salah.

💻 CONTOH KODE:

# Menyuruh Pandas membuang semua baris karyawan yang kolom gajinya kosong/hilang
tabel_bersih = tabel_karyawan.dropna()

📖 KATA 79: Data Pipeline (dibaca: de-ta paip-lain / artinya: Pipa Saluran Data)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Rangkaian skrip kodingan yang dirakit secara otomatis untuk menyedot data kotor dari satu tempat, mencucinya di tengah jalan, lalu memuntahkan data bersihnya langsung ke dalam otak AI.

💡 ANALOGI DUNIA NYATA:

Mesin Ban Berjalan (Conveyor Belt) di pabrik roti modern. Tepung dimasukkan di ujung kiri, mesin otomatis menguleninya, mengisinya dengan cokelat, memanggangnya, dan di ujung kanan sudah keluar roti dalam kemasan plastik. Koki (Engineer) tidak perlu lagi repot-repot mengangkat baskom secara manual setiap hari.

⚠️ ATURAN WAJIB:

Saluran pipa ini harus dirakit dengan sistem penanganan error (Kotak P3K / try-except) yang sangat kuat. Kalau ada satu data aneh yang menyumbat pipa, sistem tidak boleh mati total.

💻 CONTOH KODE (Ilustrasi Sederhana):

# Pipa otomatis yang berjalan berurutan
data_mentah = sedot_data_dari_internet()
data_bersih = cuci_data(data_mentah)
kirim_ke_database_ai(data_bersih)
