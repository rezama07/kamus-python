BAB 3: ATURAN RESEP

Bab 3 / Kamus Python (Logika Keputusan JIKA-MAKA)

Di Bab 1 dan 2, komputer kita masih seperti robot bodoh yang cuma jalan lurus. Di Bab 3 ini, kita akan mengajari komputer cara berpikir dan membuat keputusan sendiri berdasarkan kondisi tertentu.

📖 KATA 26: if (dibaca: if / artinya: Jika)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Perintah untuk menyuruh komputer mengecek sebuah syarat. JIKA syaratnya terpenuhi (Benar/True), maka lakukan pekerjaan ini.

💡 ANALOGI DUNIA NYATA:

Seperti satpam penjaga bioskop yang diberi aturan mutlak: "JIKA pengunjung bawa tiket, silakan buka pintunya". Satpam akan mengecek tangan pengunjung satu per satu.

⚠️ ATURAN WAJIB (BACA PELAN-PELAN):

  • 1. Ketik kata if, beri spasi, lalu masukkan syaratnya.
  • 2. JEBAKAN PEMULA: Saat mengecek syarat, Anda WAJIB menggunakan tanda sama dengan dua kali (==). Ingat, satu = artinya "memasukkan data", sedangkan dua == artinya "mengecek/bertanya".
  • 3. Tutup baris tersebut dengan tanda titik dua ( : ).
  • 4. Baris di bawahnya WAJIB MENJOROK KE DALAM (ditekan tombol Tab/Spasi). Baris yang menjorok ini adalah "pekerjaan rahasia" yang hanya akan dilakukan kalau syaratnya benar.

💻 CONTOH KODE:

bawa_tiket = True 
# Baris 1: Ini perintah memakai satu = untuk mengisi kotak
if bawa_tiket == True: 
# Baris 2: Ini pengecekan memakai dua == untuk bertanya: "Apakah benar dia bawa tiket?"
    print("Silakan masuk ke bioskop!") 
# Baris 3: Tulisan ini akan muncul karena syaratnya terpenuhi

📖 KATA 27: else (dibaca: els / artinya: Kalau Tidak)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Ini adalah "Rencana B" atau jalan buntu. Perintah ini dipanggil ketika syarat dari if di atasnya ternyata GAGAL terpenuhi (Salah/False).

💡 ANALOGI DUNIA NYATA:

Lanjutan dari satpam tadi: "JIKA pengunjung bawa tiket, buka pintunya. KALAU TIDAK bawa tiket (else), usir mereka!".

⚠️ ATURAN WAJIB:

Kata else tidak butuh syarat lagi (karena dia otomatis menampung semua sisa saringan dari if). Ketik else, langsung tempelkan titik dua ( : ). Baris di bawahnya juga wajib menjorok ke dalam.

💻 CONTOH KODE:

bawa_tiket = False 
# User ternyata tidak bawa tiket
if bawa_tiket == True:
    print("Silakan masuk!") 
# Baris ini dilewati oleh komputer
else:
    print("Maaf, silakan pulang!") 
# Layar akan memunculkan tulisan usiran ini

📖 KATA 28: elif (dibaca: e-lif / Singkatan dari: Else If atau "Atau Jika")

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Ini adalah "Rencana Cadangan" atau pilihan kedua, ketiga, keempat, dan seterusnya sebelum akhirnya menyerah ke else (jalan buntu).

💡 ANALOGI DUNIA NYATA:

Kembali ke satpam bioskop. Aturannya sekarang bertambah panjang:

  • if (JIKA) tiketnya VVIP, silakan masuk ke Pintu Merah.
  • elif (ATAU JIKA) tiketnya Reguler, silakan masuk ke Pintu Biru.
  • else (KALAU TIDAK bawa tiket sama sekali), silakan pulang!

⚠️ ATURAN WAJIB:

Posisinya WAJIB berada di tengah-tengah (di bawah if, tapi di atas else). Sama seperti if, elif juga membutuhkan syarat dan diakhiri dengan titik dua ( : ).

💻 CONTOH KODE:

tiket = "Reguler"
if tiket == "VVIP":
    print("Masuk Pintu Merah")
elif tiket == "Reguler":
    print("Masuk Pintu Biru") 
# (Karena tiketnya Reguler, komputer akan memilih jalur ini).

📖 KATA 29: Operator Pembanding (>, <, >=, <=, !=)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Alat ukur untuk mengecek apakah sebuah data lebih besar, lebih kecil, atau tidak sama dengan data lainnya.

💡 ANALOGI DUNIA NYATA:

Seperti polisi yang mengecek batas kecepatan di jalan tol. "Apakah kecepatan mobil ini Lebih Besar dari 100 km/jam?"

⚠️ ATURAN WAJIB (DAFTAR SIMBOL):

Digunakan di dalam baris if atau elif.

  • > artinya Lebih Besar.
  • < artinya Lebih Kecil.
  • >= artinya Lebih Besar atau Sama Dengan (angka batasnya dihitung lolos).
  • <= artinya Lebih Kecil atau Sama Dengan.
  • != artinya TIDAK SAMA DENGAN (Ini unik! Python memakai tanda seru dan sama dengan untuk bilang "Tidak Sama").

💻 CONTOH KODE:

kecepatan = 120
if kecepatan > 100:
    print("Anda kena tilang!")

📖 KATA 30: and (dibaca: end / artinya: Dan)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Syarat ketat. Mengharuskan SEMUA syarat harus Benar (True) agar bisa lolos. Jika ada satu saja yang salah, maka dianggap Gagal.

💡 ANALOGI DUNIA NYATA:

Syarat naik wahana Roller Coaster. Pengunjung bisa naik JIKA: (Bawa Tiket) AND (Tinggi badan di atas 150 cm).
Punya tiket tapi pendek = Ditolak.
Tinggi tapi tidak punya tiket = Ditolak. Harus dua-duanya!

⚠️ ATURAN WAJIB:

Diketik di antara dua buah syarat pada baris if atau elif.

💻 CONTOH KODE:

bawa_tiket = True
tinggi = 140
if bawa_tiket == True and tinggi > 150:
    print("Boleh Naik!")
else:
    print("Dilarang Naik!") 
# (Komputer akan memilih jalan ini karena syarat tinggi tidak terpenuhi).

📖 KATA 31: or (dibaca: or / artinya: Atau)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Syarat santai. Asalkan SALAH SATU syaratnya Benar (True), maka sudah dianggap Lolos.

💡 ANALOGI DUNIA NYATA:

Syarat masuk jalur khusus di bandara. Anda boleh lewat jalur khusus JIKA: (Penyandang Disabilitas) OR (Membawa Bayi). Anda tidak perlu menjadi disabilitas yang membawa bayi sekaligus. Punya salah satu kriteria saja sudah cukup untuk lolos.

⚠️ ATURAN WAJIB:

Diketik di antara dua buah syarat pada baris if atau elif.

💻 CONTOH KODE:

bawa_bayi = True
disabilitas = False
if bawa_bayi == True or disabilitas == True:
    print("Silakan lewat jalur khusus!") 
# (Komputer akan meloloskan ini karena salah satu syarat, yaitu bawa bayi, bernilai True).

📖 KATA 32: not (dibaca: not / artinya: Tidak / Bukan)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Pembalik fakta. Jika sebuah syarat aslinya Benar (True), diberi kata not akan berubah menjadi Salah (False), begitu pula sebaliknya.

💡 ANALOGI DUNIA NYATA:

Persis seperti Kartu Putar Balik (Reverse) di permainan kartu UNO. Atau seperti anak kecil yang sedang tantrum: kalau Anda bilang "Iya", dia akan teriak "TIDAK!". Kata ini dipakai kalau kita justru ingin mencari tahu ketidakhadiran sesuatu.

⚠️ ATURAN WAJIB:

Cukup ketik kata not tepat di depan syarat yang ingin Anda balikkan faktanya.

💻 CONTOH KODE:

hujan = True
if not hujan: 
# Cara bacanya: JIKA TIDAK hujan...
    print("Ayo main di luar!")

# (Karena aslinya sedang hujan, maka syarat "TIDAK HUJAN" menjadi Salah. Komputer tidak akan mencetak tulisan tersebut).

📖 KATA 33: in (dibaca: in / artinya: Di dalam)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Mesin pencari kilat. Menggantikan tugas kita untuk mengecek satu per satu apakah sebuah data/barang ada di dalam sebuah keranjang (List) atau di dalam sebuah teks (String).

💡 ANALOGI DUNIA NYATA:

Seperti guru yang sedang mengecek daftar absensi kelas. Ketimbang guru itu membaca seluruh nama murid dari A sampai Z, ia cukup bertanya: "Apakah nama 'Budi' ADA DI DALAM daftar ini?"

⚠️ ATURAN WAJIB:

Posisinya harus diapit: [Barang yang dicari] in [Nama Keranjangnya]. Paling sering digabungkan dengan perintah if.

💻 CONTOH KODE:

keranjang = ["Apel", "Jeruk", "Mangga"]
if "Apel" in keranjang: 
# Komputer otomatis membongkar keranjang untuk mencari Apel
    print("Yeay, kita punya Apel!")
