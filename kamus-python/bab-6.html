BAB 6: MEMINJAM ALAT TETANGGA

Bab 6 / Kamus Python (Modul & Imports)

Di dunia nyata, tidak ada koki yang membuat panci, kompor, atau kulkasnya sendiri dari nol. Mereka membelinya. Di Python, Anda juga bisa "mengambil" alat canggih (Modul/Library) yang sudah dibuat oleh programmer jenius lain agar kodingan Anda cepat selesai.

📖 KATA 43: import (dibaca: im-port / artinya: Masukkan / Impor)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

"Komputer, tolong pergi ke gudang dan ambilkan sekotak alat bernama X bawa ke ruang kerjaku sekarang."

💡 ANALOGI DUNIA NYATA:

Bayangkan Anda sedang memasak dan butuh blender, tapi Anda tidak punya. Anda pergi ke tetangga dan berkata, "Boleh saya import (pinjam) blendernya?" Setelah tetangga memberikannya, barulah Anda bisa memakai blender itu di dapur Anda.

⚠️ ATURAN WAJIB:

Perintah import selalu diletakkan di baris paling atas dari seluruh kodingan Anda. Ini aturan tata krama (PEP 8) agar komputer menyiapkan semua alatnya sebelum mulai membaca instruksi kerja.

💻 CONTOH KODE:

import math  # Mengambil sekotak alat matematika dari gudang Python

# Sekarang kita bisa pakai alat 'sqrt' (akar kuadrat) dari dalam kotak 'math'
hasil = math.sqrt(25)
print(hasil) # Layar mencetak: 5.0

📖 KATA 44: from ... import (dibaca: from ... im-port / artinya: Dari ... Ambil)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

"Komputer, dari gudang X, aku tidak butuh satu kotak penuh. Tolong ambilkan alat spesifik bernama Y saja."

💡 ANALOGI DUNIA NYATA:

Tetangga Anda punya sekotak besar perlengkapan tukang kayu. Anda cuma butuh palu. Daripada Anda menyeret satu kotak besar yang berat itu ke rumah (import tukang_kayu), Anda cukup bilang: "Dari kotak perlengkapanmu, tolong ambilkan palunya saja" (from tukang_kayu import palu). Ini membuat program Anda lebih ringan dan berlari lebih cepat.

⚠️ ATURAN WAJIB:

Formatnya harus pas: from [Nama_Kotak] import [Nama_Alat_Spesifik].

💻 CONTOH KODE:

from random import randint  # Dari kotak 'random', ambil alat pengacak angka 'randint' saja

# Kita tidak perlu lagi mengetik kata 'random' di depannya
angka_dadu = randint(1, 6)
print("Angka dadu yang keluar:", angka_dadu)

📖 KATA 45: as (dibaca: es / artinya: Sebagai / Alias)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

Mengganti nama alat yang kita pinjam dengan julukan (alias) yang lebih pendek atau lebih gampang diingat.

💡 ANALOGI DUNIA NYATA:

Anda meminjam mesin pembuat kopi dari Italia bernama "La Marzocco Linea Micra Espresso Machine". Karena kepanjangan dan lidah Anda keseleo menyebutkannya setiap saat, Anda memberi label julukan di mesin itu: "Mulai sekarang aku panggil kamu KopiMaker saja ya."

⚠️ ATURAN WAJIB:

Selalu diletakkan di akhir baris setelah perintah import.

💻 CONTOH KODE:

# Kita meminjam library pengolah data bernama pandas, lalu memberinya julukan 'pd'
import pandas as pd

# Daripada mengetik 'pandas' panjang-panjang, kita cukup memanggil julukannya
data = pd.DataFrame({"Nama": ["Budi", "Siti"]})
print(data)

📖 KATA 46: pip install (dibaca: pip instal / artinya: Beli Alat Baru)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

"Tolong pergi ke toko alat sedunia di internet, cari alat bernama X, lalu beli dan bawa pulang ke laptopku."

💡 ANALOGI DUNIA NYATA:

Di Kata 43 (import), Anda meminjam blender dari tetangga. Tapi bagaimana kalau tetangga Anda juga tidak punya blender? Anda harus membelinya dulu dari mal! pip install adalah perintah untuk membeli alat itu dari mal (internet) dan menaruhnya di gudang rumah Anda. Setelah alatnya ada di rumah, barulah Anda bisa melakukan import.

⚠️ ATURAN WAJIB (JEBAKAN FATAL):

Perintah ini TIDAK BOLEH diketik di dalam file kodingan Python Anda (yang berakhiran .py). Perintah ini harus diketik di layar hitam yang disebut Terminal/Command Prompt.

💻 CONTOH KODE:

(Diketik di dalam Terminal, bukan di file Python)
pip install pandas 

# (Komputer akan otomatis mengunduh alat pandas dari internet).

📖 KATA 47: pip list (dibaca: pip list / artinya: Cek Isi Gudang)

🗣️ ARTINYA DALAM BAHASA MANUSIA:

"Tolong catat dan sebutkan semua alat tambahan yang sudah pernah aku beli dan simpan di laptop ini."

💡 ANALOGI DUNIA NYATA:

Anda lupa apakah Anda sudah punya cetakan kue atau belum. Daripada keliling dapur mencari-cari, Anda membuka "Buku Inventaris" yang langsung menampilkan daftar semua barang yang pernah Anda beli.

⚠️ ATURAN WAJIB:

Sama seperti di atas, perintah ini hanya bisa digunakan di dalam Terminal/Command Prompt.

💻 CONTOH KODE:

(Diketik di dalam Terminal)
pip list 

# (Komputer akan menampilkan daftar panjang seperti: numpy, pandas, math, beserta nomor versinya).
