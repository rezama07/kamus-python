Selamat datang di Kamus Python dan AI Engineering!
Sebelum mulai menggunakan kamus ini, terdapat 3 ketentuan dasar yang perlu Anda pahami:

  1. Kamus ini dirancang sebagai referensi cepat (contekan), bukan untuk dihafalkan. Anda dapat membukanya kembali kapan saja saat membutuhkan bantuan.
  2. Perhatikan Ikon Peringatan (⚠️)
    Sintaksis Python sangat sensitif terhadap aturan penulisan. Bagian "Aturan Wajib" merupakan area di mana kesalahan pemrograman paling sering terjadi, sehingga perlu dibaca secara teliti.
  3. Perbedaan Huruf Besar dan Kecil (Case Sensitive)
    Dalam pandangan manusia, "Budi" dan "budi" merujuk pada identitas yang sama. Namun bagi Python, keduanya dianggap sebagai entitas yang berbeda. Pastikan Anda menulis kode program persis seperti contoh.

📋 Konsep Dasar Python

Ketentuan umum dan fondasi dasar pemrograman. Sistem komputer menggunakan tanda baca khusus untuk menginterpretasikan kode program Anda. Berikut adalah 5 konsep dasar yang paling sering digunakan:

1. Tanda Sama Dengan ( = ) → "Operator Penugasan (Assignment)"

Dalam matematika, = berarti hasil akhir. Namun dalam Python, = adalah operator penugasan (assignment) yang berfungsi untuk memasukkan atau menyimpan data ke dalam suatu variabel.

nama = "Budi"
(Penjelasan: Menyimpan nilai teks "Budi" ke dalam variabel bernama nama)

2. Tanda Koma ( , ) → "Pemisah Elemen Data"

Digunakan sebagai pemisah antarelemen data yang ditulis secara berurutan agar interpreter Python dapat mengidentifikasi setiap item secara terpisah.

["Apel", "Jeruk", "Mangga"]

3. Tanda Kutip ( " " atau ' ' ) → "Penanda Literal Teks (String)"

Penanda wajib untuk mendefinisikan tipe data teks (String) agar sistem mengidentifikasinya sebagai nilai literal teks dan bukan sebagai instruksi program.

4. Tanda Pagar ( # ) → "Catatan Penjelas (Komentar)"

Digunakan untuk menulis catatan penjelasan di dalam kode program (Comment). Semua teks setelah tanda pagar pada baris yang sama akan diabaikan oleh interpreter saat program dijalankan.

5. Tanda Kurung ( (), [], {} ) → "Struktur Wadah Data"

Python membedakan jenis wadah penyimpanan data berdasarkan bentuk tanda kurung yang digunakan:

  • [ ] (Kurung Siku) untuk struktur data bertipe daftar dinamis (List).
  • ( ) (Kurung Biasa) untuk struktur data bertipe daftar permanen yang nilainya tidak dapat diubah (Tuple).
  • { } (Kurung Kurawal) untuk struktur data bertipe pasangan kunci-nilai (Dictionary).
